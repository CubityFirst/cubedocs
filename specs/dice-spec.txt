## Roll20 Dice Feature Parity

### Core Dice Types
| Feature | Notation | Ours |
|---|---|---|
| Basic roll | `NdX` | done |
| Fate/Fudge | `NdF` | done |
| Computed dice count/sides | `(N+Y)dX` / `Nd(X+Y)` | no |
| Pool (custom faces) | `1d[2,4,6,8]` | done |
| Table (string entries) | `1d[fire,ice]` | done |

| Feature | Notation | Ours |
|---|---|---|
| Operators `+ - * / % **` | `2d6+1d4*3**2` | done |
| Functions floor/ceil/round/abs | `floor(2d6/3)` | done |
| Parentheses | `(2d6+1d4)*2` | done |

| Feature | Notation | Ours |
|---|---|---|
| Keep highest | `kh3` | done |
| Keep lowest | `kl3` | done |

| Drop lowest shorthand | `d3` | done |
| Drop highest | `dh3` | done |
| Drop lowest explicit | `dl3` | done |
| Reroll unlimited | `r<2`, `r1r3r5` | done |
| Reroll once | `ro<2` | done |

| Exploding | `3d6!`, `3d6!>4`, `3d6!3` | done |

| Sorting ascending/descending | `8d6s`, `8d6sd` | done |
| Dice matching | `2d6m`, `2d6mt`, `6d6mt3>4` | no |
